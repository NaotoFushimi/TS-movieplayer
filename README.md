TS-movieplayer
==============

    タイトル文字列を表示出来る
    フルスクリーンに出来る
    再生時間の表示をする
    再生に合わせて、シークバーを動かす
    音量設定を出来るようにする
    再生時間に合わせた処理を出来るようにする
    再生開始、途中停止、終了等のポイントにフックを出来るようにする
